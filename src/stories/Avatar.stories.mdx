# ðŸ“¢ Avatar

`Avatar` is a component for rendering a user avatar. To represent a user from a collaborating company, use the `color` prop to create a color border around the Avatar.

Avatar sizes are mapped to the `size_avatar` set of design tokens. However, you can also set the size as a number if one of the tokens doesn't work.

## Usage

```jsx
import Avatar from 'components/shared/DesignSystem/SlabTransferPortal/Avatar';

---

<Avatar title="Jean-Ralphio Saperstein" />
```

## contentBefore

If `contentBefore` is supplied, it will replace the Notice's icon.

<Canvas>
  <Story
    name="contentBefore"
    args={{
      children: "We are processing your Photos. Sit tight!",
      contentBefore: (
        <Spinner
          size="small"
          trackColor="var(--cc_color_border_default)"
          meterColor="var(--cc_color_text_subtle)"
        />
      ),
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

## contentAlign

Use `contentAlign="top"` for longer content (a headline and a paragraph, for example).

<Canvas>
  <Story
    name="contentAlign"
    args={{
      children: (
        <div>
          <h3 style={{ marginTop: 0, marginBottom: 8 }}>Some Headline</h3>
          <p style={{ margin: 0 }}>
            Lorem ipsum dolor sit amet, consectetur adipiscing elit. Nam

            rhoncus pulvinar aliquam.
          </p>
        </div>
      ),
      contentAlign: "top",
      size: "large",
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>
