# ðŸ“› Badge

`Badge` is used to display brief status-related information. For longer text, please use `Notice`.

The most common application for a `Badge` is in a list item, to display the status of an asset, event, or user, or to denote that something is new or is behind an updgraded plan paywall. `Badge` should not be pressable, but can be part of a pressable containing other items.

## Usage

```jsx
import Badge from 'components/shared/DesignSystem/SlabTransferPortal/Badges/Badge';

---

<Badge iconName="account-check">I'm a Badge</Badge>
```

## `BadgeNew` Variation
We will be including a `BadgeNew` in the folder structure to use for when you need to call out a new feature. This should always be the same size (`small`), color (`caution`), and text, so baking everything into a component (including translations for the "New" text) that extends the regular `Badge` component makes sense in this case for ease of use and DRY behavior.

```jsx
import BadgeNew from 'components/shared/DesignSystem/SlabTransferPortal/Badges/BadgeNew';

<BadgeNew />
```
