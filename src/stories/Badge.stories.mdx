# üè∑Ô∏è Badge

`Badge` is used to display brief status-related information. For longer text, please use `Notice`.

The most common application for a `Badge` is in a list item, to display the status of an asset, event, or user. `Badge` should not be pressable, but can be part of a pressable containing other items.

## Usage

```jsx
import Badge from 'components/shared/DesignSystem/SlabTransferPortal/Badge';

---

<Badge name="Jean-Ralphio Saperstein" />
```
