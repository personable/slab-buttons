# ðŸŸ¦ Button

Defaults to rendering as a `<button type="button" />`, but will render as a link if an `href` prop is provided.

## Usage

```jsx
import Button from 'components/shared/DesignSystem/SlabTransferPortal/Buttons/Button';

---

<Button color="primary" onClick={() => {}} iconName="account-multiple">
  Primary Button
</Button>
```
