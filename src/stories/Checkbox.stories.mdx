## Standard Checkbox vs. Asset Feed Checkbox

We have two checkbox styles in the web app: a standard square-shaped input, and a larger, circular input we use in the Asset Feed.

`<Checkbox />` is built to accommodate these use cases (and future checkboxes!) by exposing the following props:

- `size`: the square dimensions of the checkbox itself (the label text will increase proportionally)
- `shape`: circular or square
- `uncheckedBackgroundColor`
- `checkedBackgroundColor`
- `checkmarkColor`
- `borderColor` (visible in unchecked state only)
