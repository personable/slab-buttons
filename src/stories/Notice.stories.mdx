# Notice

- Use `<Notice />` to communicate status (something happened or is happening), or to alert the user to important information (scheduled downtime, upcoming feature changes, etc.).
- `<Notice />` accepts text or components/HTML as `children`. It supplies basic styles to format any text inside it, but any other formatting is up to you.
- `<Notice />` can have a call to action (see `contentAfter` example) or be purely informational.

The icon can be changed via the `iconName` prop. **Note: This storybook uses [an updated version of Material Design Icons](https://fonts.google.com/icons?icon.set=Material+Icons), so it won't 100% match our icon set.**

## contentBefore

If `contentBefore` is supplied, it will replace the Notice's icon.

<Canvas>
  <Story
    name="contentBefore"
    args={{
      children: "We are processing your Photos. Sit tight!",
      contentBefore: (
        <Spinner
          size="small"
          trackColor="var(--cc_color_border_default)"
          meterColor="var(--cc_color_text_subtle)"
        />
      ),
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

## contentAlign

Use `contentAlign="top"` for longer content (a headline and a paragraph, for example).

  <Story
