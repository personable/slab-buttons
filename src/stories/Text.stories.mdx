# ✍️ Text

- Sensible defaults based on `size`, with the flexibility to render as any HTML element (see code examples under props table).
- Automatically nudges sizes up 1px if `heading` font-family (Averta) is being used.
- Margins reset to `0` by default (we don't do a lot of long-form text in the app itself). Add margin when you need it via the `ccMargin` prop.
- Can be extended as a `styled-component`.

## Usage

```jsx
import Text from 'components/shared/DesignSystem/SlabTransferPortal/Text';

---

<Text size="xxxl" as="h1" ccMargin="none none l">
  This is an H1 Heading
</Text>
```

## One-Off/Custom Styles

<br />

### `color="inherit"`

In addition to our text-color design tokens, Text's `color` prop accepts `inherit`, which lets the component inherit its color from the CSS cascade.

<Canvas isColumn columns={1} withSource="open">
  <div
    className="dark-environment"
    style={{
      padding: "2rem",
      backgroundColor: "var(--cc_color_background_1)",
      color: "var(--cc_color_brand_upgrade)",
    }}
  >
    <Text size="xl" as="h1" color="inherit" display="inline">
      I am inheriting my color from my parent.
    </Text>
    &nbsp;
    <Text size="xl" as="h1" display="inline">
      But I am not.
    </Text>
  </div>
</Canvas>

### Extending as a `styled-component`

For more extensive custom styling, extend `<Text>` as a `styled-component`.

```jsx
const StyledText = styled(Text)`
  padding-bottom: var(--cc_size_spacing_s);
  border-bottom: var(--cc_size_border_width_s) solid var(--cc_color_border_default);
`;

---

<StyledText size="xxl">
  I will have a bottom border and a little extra bottom padding
</StyledText>
```
