# Text

One-stop shop for text styles on the web.

- Sensible defaults, but also flexible. For example, sizes `xxxl` and `xxl` render as `h1` with `heading` font-family by default (see code examples under props table).
- Automatically bumps sizes up 1px if `heading` font-family (Averta) is being used.
- Margin to set to `0` by default because we don't do a lot of long-form text in the app itself. Add margin when you need it via the `ccMargin` prop.
- Can be extended as a `styled-component` if you want to customize more than the props allow (add padding, add a bottom-border, etc.).
